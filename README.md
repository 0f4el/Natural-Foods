## 🚀 Como executar o projeto

### Executando com o arquivo BAT (Windows)

Para facilitar a execução da aplicação, foi criado um arquivo `iniciar.bat` que automatiza todo o processo:

1. **Verifica** se o Python está instalado
2. **Verifica** se o Flask está instalado (instala automaticamente se necessário)
3. **Configura** as variáveis de ambiente do Flask
4. **Inicia** o servidor em `http://localhost:5000`
5. **Abre** automaticamente o navegador Chrome

### Como usar

1. Dê um duplo clique no arquivo `iniciar.bat`
2. Aguarde o servidor iniciar
3. O navegador será aberto automaticamente
4. Para parar o servidor, pressione `CTRL + C` no terminal
